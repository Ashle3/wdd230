Color Scheme

Header

The heading of each webpage will have a background color of red (rgb(255, 0, 0) with the font color being white (rgb(255, 255, 255)).

Header Example

Subheadings, Secondary Color

The secondary color of the website, which will be used on subheadings such as the navigation bar, will be the color gray (rgb(128, 128, 128)), and will use white text (rgb(255, 255, 255)).

Subheading Example

Extras

Any other remaining space on the website will remain white (rgb(255, 255, 255)), and any text that is on the white background will be black (rgb(0, 0, 0)).
